请创建一个功能完整、设计精美的基于 markdown 格式的知识卡片分享应用，确保所有组件都能正常工作并具有良好的用户体验，具备以下完整功能和设计要求：

**核心功能：**

1. **仪表板页面** - 显示每日热量摄入/消耗的圆形进度指示器（目标1200卡路里），包含用户头像、个性化问候语、日期选择器和宏量营养素卡片（碳水化合物、蛋白质、脂肪）

1. **仪表板页面** - 显示当前用户创建的知识卡片，一行展示2张知识卡片。卡片包含用户头像和标题，浏览量、收藏、点赞数。
2. **卡片详情页面** - 支持模糊搜索和分类筛选
2. **食物数据库** - 内置丰富的食物数据库，包含常见食物的详细营养信息（热量、碳水、蛋白质、脂肪）
3. **食物搜索功能** - 实时搜索食物，支持模糊匹配和分类筛选
4. **食物详情页** - 显示食物图片、营养成分、份量选择和添加到餐食的功能
5. **餐食记录系统** - 支持早餐、午餐、晚餐和零食的分类记录
6. **进度追踪页面** - 显示整体进度百分比、连续打卡天数、营养和健身指标
7. **底部导航栏** - 在仪表板、搜索、进度和个人资料之间切换


**设计要求：**
- **颜色方案**：以健康的绿色为主色调（`#22c55e`, `#16a34a`），配合中性色（白色、灰色）
- **界面风格**：现代简洁的移动端设计，圆角卡片布局，清晰的视觉层次
- **交互元素**：圆形进度条、可点击的食物卡片、平滑的页面切换动画
- **图标系统**：使用 Lucide React 图标库，保持一致的视觉风格




**用户体验：**
- 直观的热量追踪界面
- 快速添加常用食物
- 清晰的营养成分展示
- 激励性的进度反馈
- 流畅的导航体验
